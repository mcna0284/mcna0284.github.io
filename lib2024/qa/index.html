5. Qus: Identify the odd one from these tags: <span>, <p>, <a>, <i> and state the reason?
Ans: The <p> is the odd tag. It is a block-level element that starts on a new line and the rest of the tags are inline elements
6. Qus: What are the tags required to create a navigation list in HTML?
Ans: <li>, <ul>, <a>
7. Qus: What element is used to create images? Name its attributes.
Ans: <img> Attributes: src and alt
8. Qus: Identify the odd one from these tags: <table>, <tr>, <b>, <td> and state the reason
Ans: The odd tag is <b>. It is the odd one because the rest of the tags are used to make a table and this odd tag is only used to make text bold.
9. Qus: Give an example of color picker input.

10. Qus: What is the use of name and value attributes? Give two examples
Ans: Name defines the value tag. It identifies information on what the value tag will be. Value is the link. Ex: <a href="url"</a>, <img src="img_girl.jpg">

12. Qus: How would you link an external style sheet? Give example.

Ans: External styles are defined within the <link> element, inside the <head> section of an HTML page. An external style sheet can be written in any text editor, and must be saved with a .css extension. The external .css file should not contain any HTML tags. Ex: <head> <link rel="stylesheet" href="mystyle.css"> </head>
